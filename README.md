# Phishing Awareness Training

## Objective
Create awareness about phishing attacks and educate users on identifying and preventing phishing attempts.

## Topics Covered
- What is Phishing?
- Common Phishing Techniques
- Warning Signs of Phishing Emails
- Prevention and Best Practices

## Learning Outcome
Understanding how phishing attacks work and how to protect against them.
